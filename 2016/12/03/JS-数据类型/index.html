JS 数据类型

Dec 03, 2016读书笔记点击

JavaScript 中的数据类型：简单数据类型 undefined, null, boolean, number和string 五种，复杂数据类型 object；通过typeof操作符来识别


1. undefined: 只有一个值的数据类型，值undefined，在使用var声明变量但未对其加以初始化时，这个变量的值就是undefined。注意：typeof(undefined) 返回也是 undefined。可以将undefined赋值给任何变量或属性，但并不意味了清除了该变量，反而会因此多了一个属性。

2. null:只有一个值的数据类型，这个特殊的值是null。从逻辑角度来看，null值表示一个空对象指针。注意：typeof(null)返回object，但null并非object，具有null值的变量也并非object。

3. boolean: 该类型只有两个字面值：true和false。这两个值与数字值不是一回事，因此true不一定等于1，而false也不一定等于0。

4. number: 这种类型用来表示整数和浮点数值，还有一种特殊的数值，即NaN。 注意：typeof(NaN)和typeof(Infinity)都返回number 。NaN参与任何数值计算的结构都是NaN，而且 NaN != NaN 。Infinity / Infinity = NaN 。

5. string:用于表示由零或多个16位Unicode字符组成的字符序列，即字符串。可以由单引号(‘)或双引号(“)表示。

6. object:对象其实就是一组数据和功能的集合。对象可以通过执行new操作符后跟要创建的对象类型的名称来创建。创建Object类型的实例并为其添加属性和（或）方法，就可以创建自定义对象
